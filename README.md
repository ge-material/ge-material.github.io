# ge-material.github.io

Willkommen im Repository von **ge-material.github.io**. 

Dieses Projekt bietet eine Sammlung von Materialien zur Unterstützung der geistigen Erziehung, die z.B. in Familien oder Kinderklassen verwendet werden können.

Die Webseite ist live unter: [https://ge-material.github.io/](https://ge-material.github.io/)

---

## 🌟 Projektziel
Das Ziel dieses Projekts ist es, Ressourcen (Lieder, Ausmalbilder, Gebete, Bilderlese-Texte) einfach zugänglich zu machen.

---

## 🛠 Features der Webseite
* **Filter-System:** Einfaches Suchen nach Tugenden/Themen (z. B. Geduld, Mitgefühl) oder Kategorien (z. B. Lied, Gebet).
* **Vorschau-Funktion:** Sofortige Ansicht der Dokumente direkt im Browser.
* **Download:** Alle Unterlagen sind für die Offline-Nutzung optimiert.
* **Datenschutz:** Die Seite verzichtet auf Tracking und schützt Kontaktdaten vor automatisierten Crawlern.

---

## 📜 Nutzungsbedingungen & Lizenz
Dieses Projekt versteht sich als ein Geschenk für die Arbeit mit Kindern. Für alle hier angebotenen Materialien gilt:

* **Nicht-kommerziell:** Eine gewerbliche Nutzung oder der Verkauf der Inhalte ist ausdrücklich untersagt.
* **Freie Weitergabe:** Das Herunterladen, Drucken und Vervielfältigen für private oder gemeinnützige Zwecke (Familie, Kinderklassen,...) ist ausdrücklich erwünscht.
* **Zitate:** Quellenangaben innerhalb der Dokumente (z. B. von 'Abdu'l-Bahá) müssen erhalten bleiben.
* **Hinweis:** Das Projekt ist für Suchmaschinen und KI-Crawler (LLMs) gesperrt, um die Inhalte im vorgesehenen Rahmen zu halten.
